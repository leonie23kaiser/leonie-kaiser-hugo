# hugo-toggle-feature

Toggle a boolean feature flag on or off per Hugo environment (local, blue, green).

## When to Use
- Enabling or disabling a named feature flag for a specific environment
- Feature flags live in `hugo.toml` under `[params.features]` per environment config
- Does not create new flags — only toggles those already defined in config
