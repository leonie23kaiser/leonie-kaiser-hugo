# hugo-update-config

Update a named configuration parameter per Hugo environment.

## When to Use
- Changing `apiBaseUrl`, `analyticsId`, or any other named param per environment
- Config params live in `hugo.toml` under `[params]` per environment config file
- Does not add new params — only updates values for existing keys
