# hugo-reflect

Analyze recent session logs and produce structured summaries of learnings and patterns.

## When to Use
- Capturing new conventions or patterns after a significant work session
- Reviewing `.conversation/raw/*.jsonl` logs to extract reusable knowledge
- Writing summaries to `.conversation/features/` or `.conversation/general/`
- Improving skills or agent instructions based on observed behaviour
