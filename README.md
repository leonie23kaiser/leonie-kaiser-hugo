# Leonie Kaiser – Hugo

Personal site for Leonie Kaiser. Shares Hugo stack and design-system with
`superleague-hugo` (same brand.css, same header/footer shape).

## Quickstart

```bash
hugo server -D
# http://localhost:1313/
```
